VI and Control Reference

Use the LabVIEW classes and associated VIs to programmatically interact with a remote SSH and/or SFTP server.

Palette Object Description
Session An object representing the underlying session resource for the libssh2 shared library.
SFTP An object representing a connection to remote SFTP server. This is the parent class used to create all other SFTP-related objects.
SSH The parent, or root, class used to create all other libssh2-related objects.
