Read Modified Time VI

Owning Palette: SFTP Attributes

Requires: LIBSSH2

Gets the last time the remote SFTP filesystem entry was modified in seconds relative to the UNIX epoch.

Modified Time is the last time the data from a file or directory was written by a process on the remote SSH server. It is encoded as a UNIX timestamp as the number of seconds since January 1st 1970 (UTC).
SFTP Attributes Out is the same object from SFTP Attributes In.
SFTP Attributes In represents a libssh2 shared library SFTP Attributes struct.
