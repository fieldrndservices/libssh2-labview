Read Access Time VI

Owning Palette: File Info

Requires: LIBSSH2

Gets the last time the remote filesystem entry was read in seconds relative to the UNIX epoch.

Access Time is the last time the data from a file was read by a process on the remote SSH server. It is encoded as a UNIX timestamp as the number of seconds since January 1st 1970 (UTC).
File Info Out is the same object from File Info In.
File Info In represents a libssh2lv shared library File Information struct.
