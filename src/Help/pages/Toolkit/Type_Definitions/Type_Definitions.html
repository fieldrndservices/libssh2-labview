Owning Palette: Type Definitions

Requires: LIBSSH2

Folder Item Description
Bit Mask Modes The bit mask value used to convert permissions to the POSIX "mode" integer value.
Modes The "rwx" groups for permissions, typically referred to as "modes" in POSIX and SFTP.
Permissions The Owner, Group, and Others permissions for a file or directory. This is modeled after the POSIX and SFTP standards.
