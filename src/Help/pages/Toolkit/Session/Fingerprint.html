Fingerprint VI

Owning Palette: Session Class

Requires: LIBSSH2

Gets a hash of the remote SSH server's host key. This is an unique identifier for the remote SSH server, typically referred to as a "fingerprint". The server's host key hash can be used to ensure the remote SSH server is valid, recognized, and geninue. If the fingerprint does not match an existing, known remote SSH server's host key hash, then do NOT proceed with authentication.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Session In is the Session object representing an initialized session in the underlying libssh2 library.
Type is the encryption to use for encoding the fingerprint.
Sesssion Out is the same Session object representing an initialized session in the underlying libssh2 library from the input.
Error Out contains error information. This provides standard error out functionality.
Fingerprint is the unique hash identifier for the remote SSH server.
