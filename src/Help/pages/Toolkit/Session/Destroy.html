Destroy VI

Owning Palette: Session Class

Requires: LIBSSH2

Cleanly and safely free the session in underlying libssh2 shared library. This must be called, or executed, after all other session-related VIs. Once this is called, no other session-related VIs should be called, or executed. This frees all internal memory used by the libssh2 shared library for this session and a segmentation fault will occur if session-related VIs are executed for this session object.

Session a Session object representing an initialized session in the underlying libssh2 library.
Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Error Out contains error information. This provides standard error out functionality.
SSH is the parent, or root, SSH object that represents the initialized underlying libssh2 library that created the session object.
