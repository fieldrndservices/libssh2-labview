Create VI

Owning Palette: Session

Requires: LIBSSH2

Creates a new Session object that represents a libssh2lv shared library Session struct. This VI must be called, or executed, before using any other Session-related VIs. This VI allocates memory within the libssh2lv shared library that must be explicitly deallocated, or freed, at the end of the application using the Destroy VI to avoid memory leaks or running Out-of-Memory (OOM) on a resource contrained system.

SSH

Create a new Session object from a SSH context.

SSH represents a libssh2-related context used to create the Session object in Session.
