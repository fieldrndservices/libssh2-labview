Create VI

Owning Palette: Session Class

Requires: LIBSSH2

Initializes a SSH session object in the underlying libssh2 library. This VI must be called, or executed, before using another session-related VIs. Typically, this should be called immediately after initializing the underlying libssh2 library. This VI allocates memory within the libssh2 library that must be explicitly deallocated at the end of the application to avoid memory leaks.

Error In Describes error conditions that occur before this node runs. This input provides standard error in functionality.
SSH is the parent, or root, SSH object that represents the initialized underlying libssh2 library.
Sesssion is a new Session object. This represents an initialized session in the underlying libssh2 library.
Error Out contains error information. This provides standard error out functionality.
