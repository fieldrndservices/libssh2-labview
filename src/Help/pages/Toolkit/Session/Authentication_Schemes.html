Authentication Schemes VI

Owning Palette: Session Class

Requires: LIBSSH2

Gets a list of authentication methods supported by the remote SSH server for a user.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Session In is the Session object representing an initialized session in the underlying libssh2 library.
Username is the username for the remote SSH server.
Sesssion Out is the same Session object representing an initialized session in the underlying libssh2 library from the input.
Error Out contains error information. This provides standard error out functionality.
Supported Schemes is all authentication schemes supported by the remote SSH server for the given username. Authentication schemes are listed in order of preference by the remote SSH server.
