VI and Control Reference

Use the VIs to safely and cleanly initialize and shutdown the underlying libssh2 shared library.

Palette Object Description
Authenticate Authenticates a session for a user with a remote SSH server.
Authenticate Schemes Gets a list of authentication schemes supported by a remote SSH server for a user.
Connect Connects a session with a remote SSH server.
Create Creates a new session object and resource in the underlying libssh2 shared library.
Destroy Destroys a session object and frees the session-related resources in the underlying libssh2 shared library.
Disconnect Disconnects a session from a remote SSH server.
Fingerprint Gets the host key hash for a remote SSH server.
Host Key Gets the host key for a remote SSH server.
Read Authenticated Determines if the session has been authenticated.
Read Mode Gets the mode of communication between the session and a remote SSH server.
Write Mode Sets the mode of communication between the session and a remote SSH server.
