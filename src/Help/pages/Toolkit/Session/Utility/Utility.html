Utility Controls and VIs

Owning Palette: Utility

Requires: LIBSSH2

Folder Item Description
Read Handle Gets the memory address for the Session struct in the libssh2lv shared library. This value should not be modified and only used with extreme caution.
Read Shared Library Gets the path to the libssh2lv shared library used to interface with a remote SSH/SFTP server.
Read SSH Gets the SSH context used to create this Session object and corresponding libssh2lv shared library Session struct.
