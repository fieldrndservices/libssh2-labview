Initialize VI

Owning Palette: SSH Class

Requires: LIBSSH2

Initialize the underlying libssh2 library. This must be called, or executed, before using any other VIs in this toolkit. This typically initializes the cryptography library (OpenSSL), which is statically compiled with the libssh2lv shared library.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
libssh2lv Shared Library is a path to the libssh2lv shared library. If <Not A Path>, then the search path will be built based on the current working directory, the source code of the project, and system-specific folders for libraries, such as the /usr/lib directory on Unix-like systems. If a valid, non-empty path is supplied, then the search path only includes this path and skips all other directories.
Error Out contains error information. This output provides standard error out functionality.
SSH is a new SSH object from the SSH LabVIEW class. This is the parent, or root, object needed to create all other objects in this toolkit.
